# Asset Files

This directory should contain:
- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

For development, you can use Expo's default assets or create your own.

To add custom assets, simply place PNG files with the names above in this directory.
